# Student Pass Predictor

## Overview

Predict whether a student will pass or fail based on:

* Study Hours per Week
* Attendance Rate
* Previous Exam Scores

## Model

* Logistic Regression
* Feature Scaling using StandardScaler
* Binary Classification

## Results

* Accuracy: 80.99%
