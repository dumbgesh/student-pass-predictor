# Student Pass Predictor

## Problem Statement

Predict whether a student will pass or fail based on:

- Study Hours per Week
- Attendance Rate
- Past Exam Scores

## Model Used

Logistic Regression

## Features

- Study_Hours_per_Week
- Attendance_Rate
- Past_Exam_Scores

## Results

Accuracy: 80.99%

Confusion Matrix:
[[58 13]
 [14 57]]

## Key Learnings

- Logistic Regression
- Sigmoid Function
- Binary Cross Entropy
- Gradient Descent
- Model Evaluation
